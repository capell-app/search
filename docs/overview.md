# Search

<!-- prettier-ignore-start -->

## What it does for you

Search powers the search box on your site. You choose what visitors can find, see what they are searching for, and add synonyms so different words find the same thing.

## Your screens

- **Search settings**: what content is searchable.
- **Search activity**: what visitors are typing into search.
- **Synonyms**: equivalent words that should return the same results.

## What you can do

- Decide which content appears in site search.
- See what visitors are searching for.
- Add synonyms so related words find the same results.
- Improve results for searches that return nothing.

## Where to find it

Go to **Search** in the admin to configure what is searchable and to review search activity.

## Good to know

- See what visitors are searching for, so you can give them what they want.
- Add a **synonym** so "tee" also finds "T-shirt".
- If a common search returns nothing, add the missing content or a synonym.
